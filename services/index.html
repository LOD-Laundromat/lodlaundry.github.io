Services

Which services can I use?

SPARQL Use the SPARQL endpoint to access all the crawling and dataset meta-data. Using SPARQL, you're able to fetch the document identifiers (MD5 hashes) which we'll use in the services below
Meta-Data download We provide a daily snapshot of all the crawling data (including references to the cleaned files) here
Dataset download To download a cleaned compressed (gzip) dataset, use its MD5, and use this url: http://download.lodlaundromat.org/<md5> (add ?hdt to download the HDT file instead of the Gzipped one)
Dataset browsing/querying To browse or query the dataset use the Triple Pattern Fragment API at http://ldf.lodlaundromat.org/<md5>. Querying is only supported for simple triple patterns
The LOD Laundromat Index Use the LOD Laundromat index to find datasets containing a particular resource, or using a particular namespace
