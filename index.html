Your Linked Data Laundry Machine!

We provide a method for crawling the Linked Data cloud, as well access to a large set of uniformly formatted datasets. These datasets are siblings of existing Linked Datasets, published as gzipped files, uniformly formatted (canonical N-Triple format), without duplicates, and without blank nodes. For more information on the advantages, and our approach, see our paper (under submission at ISWC)

Laundry Basket

Check what LOD data we wash clean. If you have any dirty laundry, feel free to drop them in the basket here! We automatically add these items to our seed list, used by the LOD Washing Machine

Washing Machine

Where we put dirty Linked Data in a washing machine, add some detergent... let the thing run for a while... and get out clean data!
